2.git status: to check the txt files that hasn't been committed yet
3. git add: to add the txt files
4. git commit -m "" : To add wording in the ""

7. git checkout : add the branch numbers,letters to temporarily delete ur files
8. git checkout master: to get back the temporarily files back
